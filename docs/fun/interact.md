# 交互

## 画画

### [Silk](http://weavesilk.com/)

> Silk 是一个对称绘画的网站。笔触非常独特，它会自动进行对称绘图，即使你没有任何绘画基础，也可以在这里画出非常好看的图形。在网页的设置中，还可以调整增加对称轴的数量、调整颜色、是否中心对称等，可以绘制出非常华丽的图形。

### [Fluid Paint](https://david.li/paint/)

> 一个网页画板工具，可以定制笔刷，产生刷油漆的流体效果，非常精美。
